# 🤖 AI Resume ATS Analyzer

> An AI-powered resume analysis system that automatically evaluates resumes using Google Gemini AI and generates ATS reports.

---

## 🚀 Overview

This project is an end-to-end AI automation workflow built using **Make.com**, **Google Gemini AI**, **Google Drive**, **PDF.co**, and **Google Docs**.

Whenever a new resume is uploaded to Google Drive, the workflow automatically:

- 📂 Detects the new resume
- 📥 Downloads the PDF
- 📄 Extracts resume text
- 🤖 Sends it to Google Gemini AI
- 📊 Generates an ATS analysis
- 📝 Creates a formatted report in Google Docs

No manual work required.
